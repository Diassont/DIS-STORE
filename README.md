# Інтернет-магазин комп’ютерних комплектуючих «DIS STORE»

Кваліфікаційна робота на підтвердження ступеня фахового молодшого бакалавра  
([ВСП «ППФК НТУ «ХПІ»](http://polytechnic.poltava.ua)).  
Керівник роботи – **Олійник Владислав Васильович**.

## Завдання до дипломної роботи

Розробити та вдосконалити веб-орієнтовану інформаційну систему у вигляді
інтернет-магазину комп’ютерних комплектуючих «DIS STORE» на базі CMS WordPress, яка забезпечує:

- керування каталогом товарів (додавання, редагування, видалення);
- структурування товарів за категоріями;
- відображення детальної інформації про товари;
- підтримку кошика та оформлення замовлень;
- адаптацію сайту до різних розмірів екранів (ПК, планшети, смартфони);
- можливість масштабування та подальшого розширення функціональності;
- зручний та сучасний користувацький інтерфейс;
- покращену швидкодію та стабільність роботи сайту.

Система повинна працювати у сучасних браузерах та бути придатною для
реального використання в сфері електронної комерції.

## Використані технології

- **WordPress** – система керування контентом (CMS) для створення та адміністрування сайту  
- **LocalWP** – локальне середовище розробки для WordPress  
- **PHP** – серверна мова програмування для роботи WordPress і плагінів  
- **MySQL** – база даних для зберігання інформації про товари та замовлення  
- **HTML5** – створення структури веб-сторінок  
- **CSS3** – стилізація та адаптивний дизайн  
- **JavaScript** – інтерактивні елементи та клієнтська логіка  
- **Duplicator** – міграція та резервне копіювання сайту  
- **GitHub** – зберігання проєкту та контроль версій
